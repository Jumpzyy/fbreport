##### PREREQUISITES
* Python 3.x 
* pip3
* git

## Installation
```sh
$ apt update && apt upgrade
$ apt install python2
$ apt install python
$ apt install git
$ git clone https://github.com/IlayTamvan/fbreport
$ cd report
$ unzip report.zip
$ python2 report.py
```
